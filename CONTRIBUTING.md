## Contributor Guidlines
### Issue Formatting
Please provide a detailed description of the issue and include error messages if available.
### Contributions
Please follow [Pep 8](https://peps.python.org/pep-0008/) when making contributions.
### Feature Documentation
Please include documentation changes in your pull requests. README additions should include step by step instructions on how to use the new feature.
### AI Generated Code
AI Generated Code is not allowed for this project, but you're free to make a Fork
